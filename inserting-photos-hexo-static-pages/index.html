在 Hexo 静态网页中插入照片

Hexo 是一个快速、简单且强大的静态博客框架，它允许你轻松地创建和管理静态网站。在撰写博文时，插入照片是一种有效的方式，可以丰富内容，吸引读者的注意力。

插入照片的基本语法

在 Hexo 中，你可以使用 Markdown 语法来插入照片。基本的语法如下：

![图片描述](图片链接)
  • 图片描述 是对图片的简短描述，这是可选的。
  • 图片链接 是图片的 URL 或者相对路径。

使用相对路径插入图片

如果你的图片存储在 Hexo 项目的 source 目录下的 images 文件夹中，可以使用相对路径插入图片。例如，如果你的图片路径是 source/images/myphoto.jpg，可以这样插入图片：

![我的照片](/images/myphoto.jpg)

使用外部链接插入图片

如果图片位于外部网站，你可以使用完整的 URL：

![外部图片](https://example.com/path/to/image.jpg)

确保图片的路径或链接是正确的，以便 Hexo 在生成静态网页时能够正确引用和显示这些图片。

测试

通过简单的 Markdown 语法，你可以轻松地在 Hexo 的静态网页中插入照片，为你的博文添加视觉吸引力。

这是一个简单的例子，你可以根据需要对博文进行扩展和个性化。希望这篇介绍对你在 Hexo 中插入照片有所帮助！
